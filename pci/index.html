The Problem ICP Solves

El Problema que Resuelve el ICP

Every system that receives signals faces the same fundamental question: how much happened during this time interval? Not in the sense of how much data arrived, but how much of that data crossed a significance threshold — how much deserved to be integrated into the system's state.

Todo sistema que recibe señales enfrenta la misma pregunta fundamental: ¿cuánto ocurrió durante este intervalo de tiempo? No en el sentido de cuántos datos llegaron, sino de cuántos de esos datos cruzaron un umbral de significancia — cuántos merecían ser integrados en el estado del sistema.

Formal Definition

Definición Formal

Given a signal stream processed by the Perceptual Wave Function Ψ(t), with L total samples in a session and collapse threshold τ, we define:

Dado un flujo de señal procesado por la Función de Onda Perceptual Ψ(t), con L muestras totales en una sesión y umbral de colapso τ, se define:

ICP = ln(1 + L · E) where E is the resonant density: E = |{t : |Ψ(t)| > τ}| / L

That is, E is the fraction of samples where the magnitude of Ψ(t) exceeds the threshold τ. The product L · E counts the absolute number of perceptual collapses in the session.

Es decir, E es la fracción de muestras en que la magnitud de Ψ(t) supera el umbral τ. El producto L · E cuenta el número absoluto de colapsos perceptuales en la sesión.

Mathematical Properties

Propiedades Matemáticas

  • Lower bound: ICP ≥ 0 always. ICP = 0 only if E = 0 (no sample crosses τ).
  • Acotación inferior: ICP ≥ 0 siempre. ICP = 0 solo si E = 0 (ninguna muestra cruza τ).
  • Logarithmic growth: The logarithm prevents unbounded growth with L.
  • Crecimiento logarítmico: El logaritmo evita crecimiento no acotado con L.
  • Dual sensitivity: Responds to changes in both L (duration) and E (collapse density), sublinearly.
  • Sensibilidad dual: Responde tanto a cambios en L (duración) como en E (densidad de colapsos), de forma sublineal.
  • Absolute scale independence: Does not depend on original signal units. Depends on τ as a domain-relative calibrator.
  • Independencia de escala absoluta: No depende de las unidades de la señal original. Depende de τ como calibrador relativo al dominio.

Relationship between Linear PCI and Canonical ICP

Relación entre PCI Lineal e ICP Canónico

The questionnaire PCI (weighted linear sum) and the canonical ICP (logarithmic) measure the same fundamental property —density of significant perceptual events— but in different contexts:

El PCI del cuestionario (suma lineal ponderada) y el ICP canónico (logarítmico) miden la misma propiedad fundamental —densidad de eventos perceptuales significativos— pero en contextos distintos:

  • Linear PCI (questionnaire): Designed for rapid observer evaluation via 4 self-assessed dimensions on a 0–10 scale.
  • PCI lineal (cuestionario): Diseñado para evaluación rápida de un observador mediante 4 dimensiones autoevaluadas en escala 0–10.
  • Canonical ICP (technical document): Designed for sampled signal processing where L is known and E can be calculated.
  • ICP canónico (documento técnico): Diseñado para procesamiento de señales muestreadas donde se conoce L y se puede calcular E.
Both converge on the same interpretation: low values indicate low perceptual density (SUPERPOSITION), high values indicate high density (COLLAPSE).
Ambos convergen en la misma interpretación: valores bajos indican baja densidad perceptual (SUPERPOSITION), valores altos indican alta densidad (COLLAPSE).

Canonical Implementation (Python)

Implementación Canónica (Python)

import numpy as np def compute_icp(signal: np.ndarray, tau: float = 0.3) -> float: """Computes the Perceptual Complexity Index (ICP).""" L = len(signal) if L == 0: return 0.0 E = np.sum(np.abs(signal) > tau) / L return float(np.log(1.0 + L * E)) def fuse_icp(icp_values: dict, weights: dict = None) -> float: """Fuses multiple ICPs into a weighted global index.""" keys = list(icp_values.keys()) if weights is None: weights = {k: 1.0 / len(keys) for k in keys} return sum(weights[k] * icp_values[k] for k in keys) # Example: ICP on astronomical data (TESS/WASP-18b) flux = np.array([1.0, 0.998, 0.997, 0.995, 0.996, 0.999, 1.0, 1.001]) icp_astro = compute_icp(flux, tau=0.30) print(f"Astronomical ICP = {icp_astro:.4f}")
